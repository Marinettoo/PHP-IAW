Introducción a PHP

¿Qué es PHP?

PHP son las siglas de "PHP: Hypertext Preprocessor". Es un lenguaje de scripting de código abierto y gratuito, muy popular, que se ejecuta en el backend (lado del servidor).

Fue creado en 1994 por Rasmus Lerdorf.

Características Principales:

  • Lenguaje de Scripting: Se ejecuta paso a paso, interpretando las instrucciones una por una.
  • Backend: Se ejecuta en el servidor web (como Apache o Nginx), no en el navegador del usuario (como JavaScript). Su trabajo es procesar datos, conectarse a bases de datos y generar HTML que luego se envía al navegador.
  • Gratuito y de Código Abierto: Cualquiera puede usarlo, modificarlo y distribuirlo sin costo.

¿Qué es un Lenguaje de Scripting?

Un lenguaje de scripting, o de guion, es un tipo de lenguaje de programación que permite escribir secuencias de instrucciones (un script) que son leídas, interpretadas y ejecutadas una por una por un intérprete, en lugar de ser compiladas a código máquina antes de ejecutarse.

Piensa en ello como seguir una receta de cocina paso a paso (interpretado) en lugar de construir una máquina que haga el plato automáticamente (compilado).

¿Cómo ejecutar PHP? (Ejemplo con XAMPP)

Para ejecutar PHP, necesitas un servidor web. Paquetes como XAMPP (para Windows, Mac, Linux) o MAMP (para Mac) instalan todo lo necesario (Apache, PHP, MySQL) fácilmente.

  1. Instala XAMPP: Descárgalo e instálalo.
  2. Inicia Apache: Abre el panel de control de XAMPP e inicia el módulo "Apache".
  3. Crea tu archivo: Escribe tu código HTML y PHP. Guarda el archivo con extensión .php (por ejemplo, `hola.php`).
  4. Guarda en `htdocs`: Debes guardar tu archivo dentro de la carpeta `htdocs`, que se encuentra dentro de la carpeta de instalación de XAMPP (ej: `C:\xampp\htdocs\`). Puedes crear subcarpetas (ej: `htdocs\mi_proyecto\hola.php`).
  5. Abre tu navegador: Escribe en la barra de direcciones `localhost/` seguido de la ruta a tu archivo.
    • Si lo guardaste en `htdocs\hola.php`, visita `http://localhost/hola.php`
    • Si lo guardaste en `htdocs\mi_proyecto\hola.php`, visita `http://localhost/mi_proyecto/hola.php`

El servidor Apache procesará el código PHP y tu navegador solo recibirá el resultado en HTML.

Sintaxis Básica de PHP

Etiquetas PHP

Para que el servidor sepa qué parte de tu archivo es código PHP, debes encerrarlo en etiquetas especiales. El código PHP puede mezclarse con HTML.

Etiqueta Estándar (Recomendada):
Es la forma más universal y recomendada.
<?php
    // Tu código PHP va aquí
?>
Etiqueta Corta (Short Tag):
Requiere que la opción `short_open_tag` esté habilitada en la configuración de PHP. A veces no es portable.
<?
    // Código PHP
?>
Etiqueta de Script:
Similar a HTML, menos común hoy en día.
<script language="php">
    // Código PHP
</script>

Comentarios

Los comentarios son líneas que el intérprete de PHP ignora. Sirven para dejar notas y explicar el código.

<?php
    // Este es un comentario de una sola línea

    # Este también es un comentario de una sola línea

    /*
      Este es un comentario
      de múltiples líneas.
      Sirve para explicaciones más largas.
    */
?>

Imprimir en Pantalla: `echo` vs `print`

Ambas se usan para enviar datos (texto, variables) al navegador. Son casi idénticas.

echo
Es ligeramente más rápido. No devuelve ningún valor. Puede tomar múltiples argumentos (separados por coma).
echo "Hola", " ", "Mundo"; // Imprime "Hola Mundo"
print
Devuelve un valor de `1`, por lo que puede usarse en expresiones (aunque es raro). Solo acepta un argumento.
print "Hola Mundo"; // Imprime "Hola Mundo"

¿Cuándo usar? En el 99% de los casos, usa `echo`. Es el estándar de facto y marginalmente más rápido.

Variables y Tipos de Datos

Variables

¿Qué es una variable?
Una variable es un contenedor para almacenar información (un número, un texto, etc.). Se identifica por un nombre único.
Estructura (Reglas):
  • En PHP, todas las variables comienzan con el símbolo de dólar (`$`).
  • El nombre debe empezar con una letra o un guion bajo (`_`).
  • No puede empezar con un número.
  • Solo puede contener caracteres alfanuméricos y guiones bajos (A-z, 0-9, y _).
  • Son sensibles a mayúsculas (`$nombre` es diferente de `$Nombre`).
Ejemplo:
<?php
    $mi_variable = "Hola"; // Válido
    $Variable2 = 100;      // Válido
    $_usuario = "Juan";    // Válido
    
    // $2variable = 5;     // Inválido (empieza con número)
?>

Tipos de Datos

PHP es un lenguaje de "tipado débil" o "dinámico", lo que significa que no necesitas declarar el tipo de dato de una variable. PHP lo infiere automáticamente.

String (Cadena)
¿Qué es? Una secuencia de caracteres (texto). Se define con comillas simples (`'`) o dobles (`"`).
Ejemplo: `$saludo = "Hola Mundo";`
Integer (Entero)
¿Qué es? Un número no decimal (positivo o negativo).
Ejemplo: `$edad = 30;`
Float (o Double)
¿Qué es? Un número con decimales.
Ejemplo: `$precio = 19.99;`
Boolean (Booleano)
¿Qué es? Representa un valor de verdad: verdadero (`true`) o falso (`false`).
Ejemplo: `$es_valido = true;`
Array (Arreglo)
¿Qué es? Una variable especial que puede almacenar múltiples valores en un solo lugar.
Ejemplo: `$colores = array("Rojo", "Verde", "Azul");`
Object (Objeto)
¿Qué es? Una instancia de una clase (Programación Orientada a Objetos). Almacena datos y métodos.
NULL
¿Qué es? Un tipo especial que representa "sin valor". Una variable es `NULL` si se le asigna `null` o si aún no se le ha asignado valor.
Ejemplo: `$dato = null;`

Funciones Útiles:

  • gettype($variable): Devuelve el tipo de la variable como un string.
  • var_dump($variable): Muestra información detallada sobre la variable (tipo y valor). Muy útil para depurar.

Constantes

¿Qué es una Constante?

Una constante es un identificador (un nombre) para un valor simple. Como su nombre indica, su valor no puede cambiarse durante la ejecución del script.

Estructura (Cómo definirlas):

Usando define():
Es la forma tradicional. Funciona en cualquier parte del código.
define("NOMBRE_CONSTANTE", "Valor de la constante");
echo NOMBRE_CONSTANTE; // Imprime "Valor de la constante"
  • Por convención, los nombres de las constantes se escriben en MAYÚSCULAS.
  • No usan el símbolo `$` como las variables.
Usando const:
Es una forma más moderna, introducida en PHP 5.3. Se usa a nivel de clase o global, pero no dentro de funciones o bucles.
const MI_CONSTANTE = "Otro valor";
echo MI_CONSTANTE;

¿Cuándo usar Constantes?

Úsalas para valores que sabes que nunca deben cambiar, como:

  • Datos de conexión a la base de datos (usuario, contraseña).
  • Valores fijos de configuración (ej: `define("TASA_IVA", 0.21);`).
  • Nombres de directorios o rutas importantes.

Un operador es un símbolo que le dice a PHP que realice una operación (matemática, de comparación, lógica, etc.) entre variables y/o valores.

Realizan operaciones matemáticas.

Asignan un valor a una variable. El básico es =.

Comparan dos valores y devuelven un booleano (true o false).

Permiten combinar múltiples condiciones.

¿Cuándo usar? Usa && y ||. Tienen mayor precedencia (se ejecutan antes) que and y or, lo cual es el comportamiento esperado en la mayoría de lenguajes y evita errores.

Estructuras de Control: Condicionales

Los condicionales permiten que tu script tome decisiones y ejecute diferentes bloques de código según si se cumple o no una condición.

if, elseif, else

¿Qué es?
Es la estructura condicional más básica. Evalúa una condición y ejecuta un bloque de código si es verdadera.
Estructura:
<?php
    $edad = 20;
    
    if ($edad >= 18) {
        // Bloque 1: Se ejecuta si $edad es 18 o más
        echo "Eres mayor de edad.";
    } elseif ($edad >= 13) {
        // Bloque 2: Se ejecuta si la 1ª es falsa Y $edad es 13 o más
        echo "Eres un adolescente.";
    } else {
        // Bloque 3: Se ejecuta si ninguna de las anteriores es verdadera
        echo "Eres un niño.";
    }
?>
¿Cuándo usarlo?
Úsalo para la mayoría de toma de decisiones.
  • Usa if solo si solo te importa el caso verdadero.
  • Usa if...else para una decisión simple de dos caminos (A o B).
  • Usa if...elseif...else para múltiples condiciones excluyentes.

Operador Ternario

¿Qué es?
Es una forma corta de escribir un if...else simple. Es un operador, no una estructura.
Estructura:
(condición) ? (valor_si_true) : (valor_si_false);
Ejemplo:
$edad = 20;
$mensaje = ($edad >= 18) ? "Mayor de edad" : "Menor de edad";
echo $mensaje; // Imprime "Mayor de edad"
¿Cuándo usarlo?
Úsalo solo para asignaciones condicionales simples, donde quieres asignar un valor u otro a una variable. No lo uses para ejecutar bloques de código complejos, ya que se vuelve ilegible.

switch

¿Qué es?
Permite comparar una variable contra múltiples valores posibles.
Estructura:
<?php
    $dia = "Lunes";
    
    switch ($dia) {
        case "Lunes":
            echo "Hoy es Lunes.";
            break; // Importante: 'break' detiene la ejecución
            
        case "Martes":
            echo "Hoy es Martes.";
            break;
            
        case "Sábado":
        case "Domingo":
            echo "Es fin de semana.";
            break;
            
        default:
            // Se ejecuta si no coincide ningún 'case'
            echo "Es otro día de la semana.";
            break;
    }
?>
¿Cuándo usarlo?
Úsalo cuando tengas un if...elseif largo que solo compara la misma variable contra diferentes valores exactos. Es más limpio y legible que muchos elseif.

Estructuras de Control: Bucles

Los bucles, o ciclos, permiten ejecutar un mismo bloque de código múltiples veces, hasta que una condición deje de cumplirse.

while

¿Qué es?
Ejecuta un bloque de código mientras la condición sea verdadera. La condición se comprueba antes de cada ejecución.
Estructura:
<?php
    $contador = 0;
    
    while ($contador < 5) {
        echo $contador; // 01234
        $contador++; // ¡Crítico! Incrementar para no crear un bucle infinito
    }
?>
¿Cuándo usarlo?
Úsalo cuando no sabes exactamente cuántas veces se ejecutará el bucle, pero sabes la condición de parada. (Ej: "mientras haya filas en la base de datos...").

do...while

¿Qué es?
Similar a while, pero la condición se comprueba después de cada ejecución. Esto garantiza que el bloque de código se ejecute al menos una vez.
Estructura:
<?php
    $contador = 10;
    
    do {
        echo "Este código se ejecuta una vez.";
        $contador++;
    } while ($contador < 5); // La condición (11 < 5) es falsa, pero ya se ejecutó
?>
¿Cuándo usarlo?
Es poco común. Úsalo solo cuando necesitas que el código se ejecute sí o sí una vez, y luego comprobar si debe repetirse.

for

¿Qué es?
Es el bucle más estructurado, ideal para repetir un bloque un número conocido de veces.
Estructura Detallada:
for (inicialización; condición; incremento) {
    // código a ejecutar
}
  • 1. inicialización: Se ejecuta una sola vez al principio del bucle. (Ej: $i = 0).
  • 2. condición: Se comprueba antes de cada iteración. Si es `true`, el bucle sigue. Si es `false`, el bucle termina. (Ej: $i < 10).
  • 3. incremento: Se ejecuta después de cada iteración. (Ej: $i++).
Ejemplo:
<?php
    // Imprime números del 0 al 9
    for ($i = 0; $i < 10; $i++) {
        echo $i;
    }
?>
¿Cuándo usarlo?
Es la mejor opción cuando sabes de antemano cuántas veces quieres repetir el bucle (ej: "repetir 10 veces", "recorrer los 50 estados").

foreach

¿Qué es?
Un bucle especial diseñado para recorrer arrays (arreglos).
Estructura Detallada:
Opción 1: Solo valores
foreach ($array as $valor) {
    // código a ejecutar con $valor
}
Opción 2: Claves y valores (para arrays asociativos)
foreach ($array as $clave => $valor) {
    // código a ejecutar con $clave y $valor
}
Ejemplo:
<?php
    $colores = array("Rojo", "Verde", "Azul");
    
    foreach ($colores as $color) {
        echo $color . " "; // Imprime "Rojo Verde Azul "
    }
    
    $capitales = array("España" => "Madrid", "Francia" => "París");
    
    foreach ($capitales as $pais => $capital) {
        echo "La capital de $pais es $capital. ";
    }
?>
¿Cuándo usarlo?
Úsalo siempre que necesites iterar sobre todos los elementos de un array. Es más fácil y seguro que usar un bucle `for` para arrays.

break y continue

break
¿Qué es? Termina la ejecución del bucle (for, while, switch) actual.
¿Cuándo usarlo? Cuando encuentras lo que buscabas y no necesitas seguir iterando. (Ej: "buscar un usuario en un array; si lo encuentras, hacer break").
continue
¿Qué es? Salta el resto de la iteración actual y pasa a la siguiente iteración del bucle.
¿Cuándo usarlo? Cuando quieres ignorar un elemento específico del bucle pero continuar con los demás. (Ej: "en un bucle de números, si el número es impar, hacer continue para procesar solo los pares").

Arrays (Arreglos)

¿Qué es un Array?

Un array es una variable especial que puede almacenar múltiples valores bajo un solo nombre. Es una colección ordenada de elementos. Cada elemento tiene una "clave" (o índice) y un "valor".

Tipos de Arrays

1. Arrays Indexados
¿Qué es? Un array donde las claves son números enteros (índices), que empiezan automáticamente en 0.
Estructura:
<?php
    // Sintaxis moderna (recomendada)
    $frutas = ["Manzana", "Pera", "Naranja"];
    
    // Sintaxis clásica
    // $frutas = array("Manzana", "Pera", "Naranja");
    
    echo $frutas[0]; // Imprime "Manzana"
    echo $frutas[2]; // Imprime "Naranja"
?>
¿Cuándo usarlo? Para listas simples donde el orden importa pero no necesitas nombres personalizados para cada elemento (una lista de usuarios, una lista de tareas, etc.).
2. Arrays Asociativos
¿Qué es? Un array donde las claves son strings (cadenas de texto) que tú defines. Se usa para almacenar pares clave-valor.
Estructura:
<?php
    $usuario = [
        "nombre" => "Juan",
        "edad" => 30,
        "ciudad" => "Madrid"
    ];
    
    echo $usuario["nombre"]; // Imprime "Juan"
    echo $usuario["edad"];   // Imprime 30
?>
¿Cuándo usarlo? Para almacenar datos estructurados, como las propiedades de un objeto (un usuario, un producto, una configuración).
3. Arrays Multidimensionales
¿Qué es? Un array que contiene otros arrays. Permite crear estructuras complejas como tablas o matrices.
Estructura:
<?php
    $alumnos = [
        [ "nombre" => "Ana", "nota" => 9 ],
        [ "nombre" => "Luis", "nota" => 7 ],
        [ "nombre" => "Sara", "nota" => 10 ]
    ];
    
    // Acceder al nombre del primer alumno
    echo $alumnos[0]["nombre"]; // Imprime "Ana"
    
    // Acceder a la nota del segundo alumno
    echo $alumnos[1]["nota"];   // Imprime 7
?>
¿Cuándo usarlo? Para almacenar colecciones de datos estructurados (ej: una lista de usuarios, donde cada usuario es un array asociativo).

Funciones de Arrays Comunes

Función Descripción
count($array)Devuelve el número de elementos en el array.
sort($array)Ordena un array indexado por valor (A-Z, 0-9).
rsort($array)Ordena un array indexado por valor en orden reverso (Z-A, 9-0).
asort($array)Ordena un array asociativo por valor, manteniendo la relación clave-valor.
ksort($array)Ordena un array asociativo por clave.
arsort() / krsort()Versiones inversas de asort y ksort.

Funciones

¿Qué es una Función?

Una función es un bloque de código reutilizable que realiza una tarea específica. Se le da un nombre y se puede "llamar" (ejecutar) en cualquier parte del script.

Ayudan a organizar el código, evitar la repetición y hacerlo más legible (principio DRY: "Don't Repeat Yourself").

Funciones Definidas por el Usuario

Estructura:
<?php
    // 1. Definición de la función
    // 'string' y 'int' son "type hints" (opcionales pero recomendadas)
    // que indican el tipo de dato esperado y devuelto.
    function saludar(string $nombre, int $edad): string {
        // Código de la función
        $mensaje = "Hola, $nombre. Tienes $edad años.";
        
        // 'return' devuelve un valor
        return $mensaje;
    }
    
    // 2. Llamada a la función
    $saludo_juan = saludar("Juan", 30);
    echo $saludo_juan; // Imprime "Hola, Juan. Tienes 30 años."
?>
  • Argumentos (o Parámetros): Son las variables que la función recibe (ej: $nombre, $edad).
  • Valor de Retorno: Es el valor que la función "devuelve" usando la palabra clave return. Si una función no tiene return, devuelve NULL.
¿Cuándo usarlo?
Siempre que tengas una tarea que vayas a realizar más de una vez (ej: validar un email, conectar a la BD, calcular un impuesto).

Ámbito de Variables (Scope)

Local
Las variables definidas dentro de una función solo existen dentro de esa función. No se puede acceder a ellas desde fuera.
Global
Las variables definidas fuera de una función no son accesibles directamente desde dentro. Para usarlas, debes usar la palabra clave global (no recomendado) o pasarlas como argumento (recomendado).
<?php
    $variable_global = "Soy global";
    
    function miFuncion() {
        // echo $variable_global; // Esto da error
        
        global $variable_global; // Forma 1 (no recomendada)
        echo $variable_global; 
        
        $variable_local = "Soy local";
    }
    
    miFuncion();
    // echo $variable_local; // Esto da error
?>

Inclusión de Ficheros

Permiten dividir tu código en múltiples archivos para organizarlo mejor (ej: un archivo para la cabecera, otro para el pie de página, otro para las funciones).

include 'archivo.php';
¿Qué es? Inserta el código del archivo. Si el archivo no se encuentra, PHP mostrará una advertencia (Warning) y el script continuará ejecutándose.
¿Cuándo usarlo? Para contenido no esencial (ej: un pie de página, un widget). Si falla, la página principal aún puede mostrarse parcialmente.
require 'archivo.php';
¿Qué es? Inserta el código del archivo. Si el archivo no se encuentra, PHP mostrará un error fatal (Fatal Error) y el script se detendrá.
¿Cuándo usarlo? Para contenido esencial sin el cual la página no puede funcionar (ej: conexión a la base de datos, un archivo de funciones críticas).
include_once y require_once
¿Qué es? Hacen lo mismo que sus contrapartes, pero se aseguran de que el archivo se incluya solo una vez. Si intentas incluirlo de nuevo, PHP lo ignorará.
¿Cuándo usarlo? Es la opción más segura y recomendada, especialmente para archivos de funciones o clases, para evitar re-definiciones y errores.

Cadenas (Strings)

¿Qué es un String?

Un string, o cadena, es una secuencia de caracteres (letras, números, símbolos) usada para representar texto. En PHP, se definen con comillas simples (') o dobles (").

  • Comillas Simples ('): Tratan el texto de forma literal. No interpretan variables. ('$nombre' imprime literalmente `$nombre`).
  • Comillas Dobles ("): Interpretan (expanden) las variables que hay dentro. ("$nombre" imprime el valor de la variable `$nombre`).

Funciones de Strings Comunes

PHP tiene cientos de funciones para manipular strings. Aquí están las más usadas:

Función Descripción (¿Qué es?) Ejemplo
strlen($str) Devuelve la longitud (número de caracteres) de la cadena. 
strlen("Hola") // 4
str_word_count($str) Cuenta el número de palabras en la cadena. 
str_word_count("Hola Mundo") // 2
strrev($str) Invierte la cadena. 
strrev("Hola") // "aloH"
strpos($str, $sub) Busca la posición de la primera aparición de una subcadena ($sub) dentro de $str. (Devuelve false si no la encuentra). 
strpos("Hola Mundo", "Mun") // 5
str_replace($b, $r, $str) Reemplaza todas las apariciones de $b (buscar) por $r (reemplazar) en $str. 
str_replace("Mundo", "PHP", "Hola Mundo") // "Hola PHP"
explode($del, $str) Divide un string en un array usando un delimitador ($del). 
explode(" ", "Hola Mundo") // ["Hola", "Mundo"]
implode($glue, $arr) Une los elementos de un array ($arr) en un string, usando un "pegamento" ($glue). (También join()). 
implode(" ", ["Hola", "Mundo"]) // "Hola Mundo"
